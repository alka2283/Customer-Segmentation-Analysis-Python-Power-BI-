# Customer-Segmentation-Analysis-Python-Power-BI

## Project Overview

This project demonstrates an end-to-end customer analytics workflow using Python and Power BI to identify high-value customers based on their purchasing behavior.

The solution applies RFM (Recency, Frequency, Monetary) analysis on customer purchase data to segment customers into actionable business groups and provide interactive insights through a Power BI dashboard.

The project simulates a real-world business scenario where sales and marketing teams need to identify loyal customers, improve retention, and increase customer engagement.

---

## Project Objective

The goal of this project is to convert raw transaction data into actionable business insights that help organizations understand:

* Customer purchase behavior
* Customer loyalty patterns
* High-value customer identification
* Customer retention opportunities
* Regional and territory-level performance

---
## Project Overview

This project follows a complete analytics workflow, including:

* Data Preparation & Processing (Python)
* Data cleaning and preprocessing using Pandas
* Date conversion and transaction validation
* Customer eligibility analysis (Monthly Purchase ≥ 50 Bags)
* Feature engineering and aggregation
* RFM metric calculation

---

## Customer Segmentation (Python)

Implemented RFM analysis using:

* Recency (How recent a customer purchase with the company)
* Frequency (What is the frequency of customer to buy with company)
* Monetary (Total Volume bought by the customer)
* Weighted RFM Scoring (Based on the business decision)

Customer Segments:

* Loyal
* Stable
* Opportunist
* At Risk

## Visualization & Insights (Power BI)

Built an interactive Power BI dashboard featuring:
